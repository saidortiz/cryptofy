Privacy Policy

Accounts

Used for permissions that are associated with accessing social media accounts. This permission allows this application to use social media accounts such as Google Plus, Facebook or Twitter for a simple login process into Cryptofy. The account created by Cryptofy can then be used to collect coin holdings that are related to the user.

Storage

Used for permissions that are associated with accessing device local storage. This permission allows this application to use local storage on an android device in order to store simple preferences/settings for the app such as currency preference, style theme, and fingerprint access.

Fingerprint

Used for permissions that are associated with accessing biometric fingerprint scanning. This permission allows this application to use the fingerprint on an android device in order to secure access to the user's portfolio if the user deems it to be sensitive.
